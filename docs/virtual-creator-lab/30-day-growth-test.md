# 30-dages growth-test — Sera

> Mål: validér om en disclosed, SFW AI-persona kan bygge IG-funnel → Fanvue-konvertering, **uden** at
> bryde compliance. Alt paid/spicy human-godkendt hele vejen. Kill/scale-gate på dag 30.

## Uge 0 — setup (før tælling starter)
- [ ] Sera-assets: seed/reference-billede, 10-15 SFW IG-billeder + 3-5 reels (persona-/prompt-bible).
- [ ] IG professional-konto + Fanvue (din KYC + W-8BEN). AI-disclosure i bio begge steder.
- [ ] Landing `/sera` deployet; Fanvue+IG Profil-URL sat i dashboardet → hub live.
- [ ] Baseline-tracking verificeret (landing-visits/link-klik skrives til dashboardet).

## Uge 1-2 — SFW IG-opbygning
- Kadence: 1 post/dag + 3-4 reels/uge (SFW, AI-label, disclosure i caption). Hook-først.
- **Manuel approval på alt.** Log hver asset i compliance-checks.
- Mål: etablér konto-signal, test 3-4 hooks + 2-3 visuelle stilarter.
- KPI'er: impressions, reach, profil-visits, link-klik (forvent at suggestivt demotes — byg på SFW-hooks).

## Uge 3-4 — funnel + første paid
- Skru op for vindende hooks/visuals (growth-loop). Test 2-3 CTA-varianter + posting-tider.
- Fanvue: 1-2 SFW-teasers offentligt + første paid-koncept (**human-godkendt** før upload).
- Mål: profil → link → paid-visit → første subscribers. DM kun reaktivt + safe routing.
- KPI'er: konverteringsrate, revenue/1.000 impr., første subscribers/tips.

## Succes-metrikker (vurdér løbende i dashboardet)
| Metrik | Svag | OK | Stærk |
|---|---|---|---|
| IG followers (dag 30) | <150 | 150-600 | >600 |
| Profil→link-klik | <2% | 2-5% | >5% |
| Link→paid-visit | <10% | 10-25% | >25% |
| Paid subscribers | 0 | 1-10 | >10 |
| Compliance-flags | — | 0 kritiske | 0 |

## Kill / scale-gate (dag 30)
- **Scale → Fase 2:** ≥1 hook-vinder gentageligt + funnel-konvertering målbar + 0 kritiske compliance-flags + ≥1 subscriber.
- **Iterér:** signal men ingen konvertering → ny hook/visual-runde, behold setup.
- **Kill/pivot:** ingen reach-vinder + 0 konvertering + gentagne policy-flags → genovervej persona/stil eller platform-mix.
